如你所见，这是一个用于反映 Windows 系统各版本情况实时进展的项目。  
  
这是一个基于 Vue 3 的练手项目 ~~（为自己代码质量低找借口）~~ ，同时是个便民项目。由于目前在测试阶段，所以会有很多可能出现的 bug，敬请谅解~
  
已经实现的功能：
* UI 整体完工
* 从 JSON 文件中动态取得数据
* 适配移动端

正在计划实现的功能：
* 实时从 UUP 获取最新版本信息 ~~（这可能需要很长时间）~~
* 国际化，即 i18n
